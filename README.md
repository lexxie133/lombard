# АИС «Ломбард-Финанс»

Автоматизированная информационная система для учёта клиентов, залогов, оценки товаров, контроля сроков возврата и автоматического перехода невыкупленных товаров в собственность ломбарда.

## Основные функции
- Регистрация клиентов и хранение паспортных данных
- Оценка залогов и расчёт суммы займа
- Контроль сроков возврата
- Автоматическое формирование договоров
- Учёт дополнительных разовых работ (оценка, инвентаризация, реализация)

## Технологии
- Языки: C#, Python
- БД: PostgreSQL
- ОС: Windows 10/11, Windows Server
